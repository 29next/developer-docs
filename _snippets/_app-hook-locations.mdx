| App Hook Location | Description |
| ------- | ------- |
| `global_header` | Used for adding javascript and styles globally to the header of a theme.<br/>Should be loaded globally in the header, most likely in `layout/base.html`.   |
| `global_footer` | Used for adding javascript globally to the footer of a theme. <br/>Should be loaded globally in the footer, most likely in `layout/base.html`.  |
