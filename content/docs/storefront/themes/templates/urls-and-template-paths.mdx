Ensure your template paths match with expected template paths for built-in storefront views. Use the public themes on [Github](https://github.com/NextCommerceCo/) as a reference guide and starting point. All URL paths are automatically localized to the users language following your store's Localization settings.

### Localization

These are POST action endpoints used in forms for switching language, currency, or storefront geo.

| URL Name | Method | Description |
| --- | --- | --- |
| set_language | POST | Change the active language. |
| core:set-currency | POST | Change the active currency. |
| core:set-storefront | POST | Change the active storefront geo (country, language, currency). |
