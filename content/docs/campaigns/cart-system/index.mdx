The Cart System connects visitor actions on the page to the cart. Buttons add and remove packages, the cart state updates automatically, and any element on the page can reflect live cart values.

## How It Works

1. A visitor clicks a button or toggle on the page
2. The SDK adds or removes the package from the cart
3. Every element bound to cart state — totals, quantities, conditionals — updates instantly
