The Bundle Selector lets visitors pick from a set of bundle options, such as "Starter", "Value Pack", or "Best Deal". Each bundle contains a fixed list of products and quantities. When a visitor selects a bundle, the cart updates instantly. The previous choice is replaced in one step, without affecting anything else in the cart.

## How It Works

1. The page shows one or more **bundle cards**, each representing a different offer
2. The visitor clicks a card to select it. The cart updates immediately
3. If a product in the bundle has options like color or size, the visitor picks them on the card
4. Prices are shown on load and update automatically when the visitor changes a variant or currency
